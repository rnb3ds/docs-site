# Справочник API

HTTPC предоставляет 27 функций параметров запросов, 5 предустановок конфигурации, 8 встроенных middleware и полную поддержку загрузки.

## Основная архитектура

```text
Пакет httpc
├── Интерфейс Client - основной клиент, поддерживающий все HTTP-методы
├── Интерфейс DomainClienter - клиент с областью видимости домена, встроенное управление сессиями
├── Config - система конфигурации (тайм-ауты/соединения/безопасность/повторы/промежуточное ПО)
├── RequestOption - 27 функций параметров запросов
├── MiddlewareFunc - цепочка промежуточного ПО
├── Result - результат ответа (включая метаданные запроса)
└── Функции уровня пакета - используются без создания клиента
```

## Быстрый справочник

### Создание клиента

```go
client, err := httpc.New()                    // конфигурация по умолчанию
client, err := httpc.New(httpc.SecureConfig()) // безопасная предустановка
client, err := httpc.New(customConfig)         // пользовательская конфигурация
```

### Отправка запросов

```go
// Функции уровня пакета
result, err := httpc.Get(url, options...)

// Методы клиента
result, err := client.Get(url, options...)

// С контекстом
result, err := client.Request(ctx, "GET", url, options...)
```

### Обработка ответов

```go
result.StatusCode()           // код состояния
result.Body()                 // тело ответа (строка)
result.RawBody()              // тело ответа (байты)
result.Unmarshal(&data)       // разбор JSON
result.IsSuccess()            // является ли 2xx
result.Meta.Duration          // время выполнения запроса
result.Meta.Attempts          // количество повторных попыток
defer httpc.ReleaseResult(result) // освобождение в пул объектов
```
