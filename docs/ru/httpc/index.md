# HTTPC

Безопасная библиотека HTTP-клиента с безопасностью по умолчанию, встроенными интеллектуальными повторными попытками, цепочкой промежуточного ПО и повторным использованием пула объектов.

## Возможности

- **TLS 1.2+** — принудительная минимальная версия TLS, по умолчанию TLS 1.2–1.3
- **Защита от SSRF** — блокировка соединений с приватными IP по умолчанию, настраиваемые исключения CIDR
- **Интеллектуальные повторные попытки** — экспоненциальная задержка с джиттером, настраиваемая стратегия повторов
- **Управление пулом соединений** — высокопроизводительное повторное использование соединений, поддержка HTTP/2
- **Цепочка промежуточного ПО** — встроенные middleware для логирования, аудита, метрик, восстановления, ID запроса
- **Загрузка файлов** — возобновление загрузки, обратный вызов прогресса, проверка контрольной суммы
- **DNS-over-HTTPS** — встроенное разрешение DoH, снижение риска перехвата DNS
- **Повторное использование пула объектов** — встроенный sync.Pool для снижения аллокаций памяти и нагрузки на GC

## Установка

```bash
go get github.com/cybergodev/httpc
```

## Быстрый старт за 30 секунд

```go
package main

import (
    "fmt"
    "github.com/cybergodev/httpc"
)

func main() {
    result, err := httpc.Get("https://httpbin.org/get")
    if err != nil {
        panic(err)
    }
    defer httpc.ReleaseResult(result)

    fmt.Println(result.StatusCode()) // 200
}
```

## Ключевые концепции

HTTPC предоставляет три варианта использования, от простого к гибкому:

```text
Функции уровня пакета    Экземпляр клиента               Клиент домена
httpc.Get()  →  client, _ := httpc.New()  →  dc, _ := httpc.NewDomain(url)
Одноразовый запрос     Пользовательская конфигурация    Управление сессией/Cookie
```

### Предустановки конфигурации

| Предустановка | Сценарий применения |
|---------------|---------------------|
| `DefaultConfig()` | Общие сценарии, безопасные значения по умолчанию |
| `SecureConfig()` | Чувствительные к безопасности сценарии, строгие таймауты |
| `PerformanceConfig()` | Высокая пропускная способность, большой пул соединений |
| `TestingConfig()` | Тестовая среда, отключены проверки безопасности |
| `MinimalConfig()` | Лёгкие скрипты, без повторов и редиректов |
