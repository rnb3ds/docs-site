# CyberGo ドキュメントについて

## ドキュメントサイト

CyberGo ドキュメントサイトは [CyberGoDev](https://github.com/cybergodev) オーガニゼーションの統合技術ドキュメントセンターであり、すべての Go オープンソースライブラリについて日本語、英語、中国語、韓国語、ロシア語の 5 つの言語で包括的なドキュメントを提供しています。

このサイトは、開発者が各ライブラリの機能、API、ベストプラクティスを迅速に理解し、本番環境で効率的かつ安全に利用できるよう支援します。

## ドキュメント対象

| プロジェクト | 説明 | GitHub |
|-------------|------|--------|
| **JSON** | 高性能、スレッドセーフな JSON 処理ライブラリ | [cybergodev/json](https://github.com/cybergodev/json) |
| **JWT** | わずか 3 つの関数ですべての JWT 操作を処理するプロダクション級 JWT ライブラリ | [cybergodev/jwt](https://github.com/cybergodev/jwt) |
| **HTTPC** | TLS 1.2+ および SSRF 防護を備えた最新の高性能 HTTP クライアント | [cybergodev/httpc](https://github.com/cybergodev/httpc) |
| **HTML** | インテリジェントな記事認識によるプロダクション級 HTML コンテンツ抽出 | [cybergodev/html](https://github.com/cybergodev/html) |
| **DD** | 毎秒 300 万件以上を処理する高性能ロギングライブラリ | [cybergodev/dd](https://github.com/cybergodev/dd) |
| **ENV** | ゼロ依存設計のプロダクションレディ環境変数管理ライブラリ | [cybergodev/env](https://github.com/cybergodev/env) |

## 貢献する

ドキュメントの品質向上に役立つあらゆる形の貢献を歓迎します：

### ドキュメントの編集

各ドキュメントページの右上には「GitHub でこのページを編集」ボタンがあります。クリックして直接編集し、Pull Request を送信してください。

### 問題の報告

ドキュメントに誤りや不足を見つけた場合は、以下の方法でフィードバックしてください：

- [ドキュメントの問題を報告](https://github.com/cybergodev/docs-site/issues/new?template=doc-issue.md) — ドキュメントのバグや改善提案を送信
- [GitHub Discussions](https://github.com/cybergodev/docs-site/discussions) — 質問、議論、経験の共有

### 貢献フロー

1. [docs-site](https://github.com/cybergodev/docs-site) リポジトリを Fork
2. 変更用の新しいブランチを作成
3. 変更内容を説明する Pull Request を送信
4. レビューとマージを待つ
